# Grapple Script

Grapple is a programming language which is easy to learn and easy to use!

## Installation

Installation is not available currently!

## Usage

```python
# Set Variable
set {x} to 1

# Write to Console
say "Something"

# Using colors:
say "&aI'm green"

# Add to variable and remove from a variable
add 1 to {x}
remove 1 from {x}

# Read from console
ask "How's your day? "

# Write variable to console:
say "Variable X is %{x}%"

# If, ELse if and else:
if {x} is 0:
    say "X is 0!"
else if {x} is 1:
    say "X is 1!"
else:
    say "X is not 0 or 1!"

# Loop X times:
loop 100 times:
    say "Counting: %loop-value%"
```
